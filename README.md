IoTech Demo Project

This is a Next.js project created as a demo application for IoTech Company to showcase web functionality, UI components, and dynamic features.

The app demonstrates a modern, responsive interface with multi-language support, dynamic content, and integration-ready components.

## Getting Started

To run this project locally, follow these steps:

Clone the repository

```bash
git clone https://github.com/abida-parveen/io-tech-demo.git
cd io-tech-demo
```
Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

Project Features

Built with Next.js (React framework for production).
Multi-language support with English and Arabic translations.
Responsive UI components using Tailwind CSS.
Dynamic pages for services.
